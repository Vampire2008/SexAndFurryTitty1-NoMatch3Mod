# Мод убирающий 3-в-ряд из игр Sex and Furry Titty (1, 2, 3) и Русы не против ящерок (1, 2)

Данный мод включает режим истории даже если он заблокировани и не виден. Также упрощает 3-в-ряд для победы за 1-2 шага, если необходимо их сыграть.

## Установка

Скачайте архив для нужной игры из секции Release и разархивируйте его в папку с игрой.

Примечание: у каждой игры своя папка:

* SaFT 1 - `Love Stories FT`
* SaFT 2 - `Sex and the Furry Titty 2`
* SaFT 3 - `Sex and the Furry Titty 3`
* Русы не против ящерок - `Lizardwomen`
* Русы не против ящерок 2 - `Lizardwomen 2`

Данный мод поставляется вместе с Bepinex engine.
